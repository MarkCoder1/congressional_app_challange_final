## 🚀 How to Use

### 1. Import the System

```typescript
import { getPresetsForSubject, getPresetData } from "@/lib/learningMapPresets";
import { PresetRenderer } from "@/components/preset-renderer";
```

### 2. Load Presets for Subject

```typescript
const presets = getPresetsForSubject("Math");
// Returns 6 math presets
```

### 3. Render Component

```typescript
<PresetRenderer
  subject="Math"
  taskTitle="Quadratic Functions"
/>
```

### 4. That's It!

The component displays all available presets, user selects one, and the appropriate visual map renders.
